# SymbioticAI-ISD-X
A multi-dimensional Reinforcement Learning framework for social media feed optimization. It moves beyond click-based engagement to integrate cognitive load balancing, ethical guardrails, and systemic health metrics using the GPT-DNA 15D architecture.

## Installation

```bash
# Clone the repository
git clone https://github.com/Grokenthusiast/SymbioticAI-ISD-X.git

# Navigate into the directory
cd SymbioticAI-ISD-X

# Install dependencies
pip install -r requirements.txt
```

## Quick Start

```bash
# Run the simulation
python .py
```
